Checklist para funcionar

1. Crear app real

Pasar esta maqueta a Next.js, React, TypeScript y TailwindCSS con componentes separados.

2. Login

Conectar Google, GitHub, Apple y email usando Supabase Auth.

3. Base de datos

Crear tablas reales para usuarios, preguntas, respuestas, votos, comentarios, reputacion y reportes.

4. IA

Conectar OpenAI para respuesta inicial, resumen, traduccion, clasificacion, moderacion y preguntas similares.

5. Seguridad

Activar RLS, rate limits, validacion con Zod, auditoria y filtros contra spam.

6. Despliegue

Subir a GitHub, desplegar en Vercel, conectar dominio y proteger con Cloudflare.
