# Diferenciacion y monetizacion

## Que hace diferente a esta red social

Esta plataforma no compite copiando a Facebook, X, Instagram, Reddit o Quora.

La diferencia principal:

> No es una red para publicar vida personal. Es una red para convertir preguntas en conocimiento vivo.

## Principios de diferenciacion

- Todo empieza con una pregunta.
- No hay estados personales como centro del producto.
- No hay videos sin proposito.
- No hay bailes ni contenido de entretenimiento vacio.
- No hay feed infinito disenado solo para adiccion.
- La IA responde primero.
- La comunidad mejora.
- Los expertos validan.
- El conocimiento queda guardado y reutilizable.
- Cada pregunta puede servir a millones de personas despues.

## Producto simple

La aplicacion debe sentirse antigua en simplicidad, pero moderna en tecnologia.

Como Google al principio:

- Una caja grande.
- Una pregunta.
- Una respuesta.
- Todo rapido.

Como una biblioteca moderna:

- Preguntas organizadas.
- Respuestas claras.
- Categorias.
- Busqueda.
- Guardados.
- Reputacion.

Lo que NO debe tener en la primera etapa:

- Historias.
- Reels.
- Lives.
- Juegos.
- Demasiados botones.
- Feed personal complicado.
- Mensajeria privada compleja.
- Marketplace desde el primer dia.

Primero puede ser PWA:

- Se abre en navegador.

- Menor costo inicial.
- Mas facil de mantener.

Despues:

- App iOS.

## Operacion sin depender de muchos humanos

La meta es que el sistema funcione casi solo, con IA y automatizacion.

La IA puede:

- Responder preguntas.
- Traducir.
- Resumir.
- Detectar duplicados.
- Clasificar categorias.
- Moderar contenido peligroso.
- Detectar spam.
- Sugerir fuentes.
- Crear resumenes diarios.
- Ayudar al usuario a formular mejores preguntas.

Humanos necesarios al principio:

- Fundador/producto.
- Desarrollador.
- Moderador parcial.
- Expertos voluntarios o pagados despues.

La aplicacion debe evitar trabajo humano repetitivo.

## Acceso global con Cuba gratis

La marca no debe parecer de un pais especifico. Debe ser mundial.

Pero la estrategia puede tener una ventaja especial:

> Usuarios con numero de telefono cubano verificado tienen acceso gratis.

La razon:

- Cuba puede ser un mercado de adopcion fuerte.
- Muchas personas tienen hambre de Internet nuevo, conocimiento, aprendizaje y oportunidades.
- Si el producto funciona bien ahi, puede crecer por comunidad y recomendacion.
- La plataforma gana usuarios, contenido, preguntas y cultura.

Regla inicial:

- Numero +53 verificado: acceso gratuito.
- Resto del mundo: plan gratis limitado + membresia.

Importante:

- Esto no significa que la app sea cubana.
- La app es global.
- Cuba es una estrategia de acceso y crecimiento.

## Como genera dinero

La plataforma no debe depender solo de publicidad.

Debe tener varias vias de ingresos.

### 1. Membresia global

Usuarios pagan mensual por:

- Mas preguntas con IA.
- Respuestas mas profundas.
- Modelos de IA mejores.
- Historial avanzado.
- Guardados ilimitados.
- Traducciones ilimitadas.
- Resumenes personalizados.
- Modo estudio.
- Exportar respuestas a PDF.
- Carpetas privadas.

Ejemplo simple:

- Free: limitado.
- Cuba Access: gratis con telefono cubano verificado.
- Global Plus: 4.99-9.99 USD/mes.
- Global Pro: 14.99-19.99 USD/mes.

La membresia no debe complicarse. Debe sentirse como una recarga simple:

- Pagas.
- Se activa tu cuenta.
- Tienes mas IA y mas capacidad.
- Si no pagas, sigues con funciones basicas.

### 2. Creditos de IA

El usuario compra creditos para:

- Preguntas largas.
- Investigaciones profundas.
- Generar codigo.
- Crear imagenes.
- Comparar fuentes.
- Crear documentos.

Esto ayuda a controlar costos de OpenAI.

### 3. Expertos verificados

Los expertos pagan o la plataforma cobra comision por:

- Verificacion profesional.
- Perfil destacado.
- Consultas pagadas.
- Respuestas premium.
- Sesiones privadas.

Modelo:

- La plataforma cobra 10%-25% de comision.

### 4. Empresas verificadas

Empresas pagan por:

- Perfil oficial.
- Respuestas oficiales.
- Soporte publico inteligente.
- Base de conocimiento para clientes.
- Analitica de preguntas frecuentes.
- API privada.

Esto puede ser el ingreso mas fuerte a largo plazo.

Ejemplo:

- Empresa pequena: 49 USD/mes.
- Empresa mediana: 199 USD/mes.
- Enterprise: precio personalizado.

### 5. Educacion

Universidades, profesores y academias pagan por:

- Comunidades privadas.
- Cursos.
- Certificaciones.
- Evaluaciones.
- Tutores IA.
- Espacios para estudiantes.

### 6. API

Desarrolladores y empresas pagan por usar:

- Busqueda de conocimiento.
- Respuestas verificadas.
- Traducciones.
- Resumenes.
- Clasificacion de preguntas.
- Base de conocimiento sectorial.

### 7. Publicidad y patrocinios con cuidado

Publicidad solo si no destruye la confianza.

Permitido:

- Patrocinios por categoria.
- Herramientas recomendadas.
- Cursos relevantes.
- Anuncios educativos.
- Empresas verificadas que respondan preguntas de su area.
- Libros, software, servicios y productos relacionados con la pregunta.

Evitar:

- Anuncios invasivos.
- Feed lleno de publicidad.
- Venta agresiva de datos.
- Anuncios que parezcan respuestas organicas.

Modelo sencillo:

- Una marca paga por aparecer en una categoria.
- Ejemplo: una academia patrocina Programacion.
- Ejemplo: una empresa de software patrocina IA.
- Ejemplo: una universidad patrocina Educacion.
- El anuncio aparece como patrocinio claro, no como trampa.

Esto puede generar dinero sin llenar la app de basura.

## Como funcionaria tecnicamente la membresia

### Registro

1. Usuario crea cuenta.
2. El sistema pide telefono opcional para beneficios.
3. Si el numero empieza con +53 y se verifica por SMS/WhatsApp, se activa `Cuba Access`.
4. Si no, queda en `Global Free`.

### Pago

1. Usuario elige Plus o Pro.
2. Stripe Checkout procesa el pago.
3. Stripe manda webhook.
4. La base de datos actualiza el plan del usuario.
5. La app desbloquea limites.

### Limites

Ejemplo:

- Cuba Access: gratis, pero con limites diarios para controlar costos.
- Global Free: 5 preguntas IA por dia.
- Plus: 100 preguntas IA por mes.
- Pro: 500 preguntas IA por mes.
- Creditos: compra extra cuando se acaban.

Esto evita que el costo de IA destruya el negocio.
