## Reglas Innegociables

1. Auditar antes de modificar
2. Conservar todo lo que ya funcione
3. No borrar archivos sin justificación
4. No exponer claves, secretos ni credenciales
5. Usar variables de entorno + `.env.example`
6. No inventar integraciones no configuradas
7. No confirmar pagos automáticamente
8. No activar intercambio financiero P2P
9. No activar compra/venta de USDT
10. No activar blockchain ni MSM Coin
11. No instalar APK silenciosamente
12. No recolectar ubicación sin consentimiento
13. No copiar código/marca de Zapya
14. No depender de un único dispositivo o admin
15. No presentar respuestas IA como palabra de Dios
16. ELIANA no emite diagnósticos médicos, sentencias legales ni confirmaciones financieras
17. Toda operación: idempotente y auditable
18. Todo módulo nuevo incluye pruebas
19. Cada fase cierra con informe técnico
