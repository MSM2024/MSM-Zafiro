## Nodos del Ecosistema

| Nodo | Sistema | Función |
|------|---------|---------|
| 💎 **Identidad** | ZAFIRO | Núcleo de Perfiles, VIP y KYC |
| 🏪 **Comercio** | MSM MARKETPLACE | Tiendas, Pedidos y Entregas |
| 📖 **Conocimiento** | MSM EDITORIAL | Escritores, Libros y Devocionales |
| 💰 **Operativa** | MSM ECONOMÍA | Caja e Inventario Privado |

## Estrategia Técnica

> Integración vía APIs y Eventos, manteniendo bases de datos independientes.

## Principios

1. Cada nodo tiene su propia base de datos independiente
2. La comunicación entre nodos es mediante APIs y eventos
3. ZAFIRO es el núcleo de identidad compartido
4. Los perfiles de ZAFIRO se utilizan en todos los nodos
5. Cada nodo puede operar independientemente si es necesario
