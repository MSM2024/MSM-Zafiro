# ZAFIRO OS — Fases

Aplicación web, PWA instalable, launcher de módulos, dashboard, perfil, aplicaciones internas, notificaciones, almacenamiento local seguro, sincronización, búsqueda universal, ELIANA integrada.

**Estado**: 🟡 En progreso (140 rutas, 0 build errors)

## Fase 2: ZAFIRO Desktop Shell

Experiencia tipo escritorio: barra superior, dock, ventanas, aplicaciones, buscador, centro de control, notificaciones, archivos, ajustes, widgets, multitarea visual.

**Estado**: 🔴 No iniciado

## Fase 3: ZAFIRO Desktop App

Empaquetado con Tauri 2: instalador Windows, icono ZAFIRO, actualización controlada, almacenamiento local cifrado, integración con archivos, notificaciones del sistema, enlaces profundos.

**Estado**: 🔴 No iniciado

## Fase 4: ZAFIRO Mobile Shell

Launcher móvil: Android, PWA, accesos a módulos, modo offline, notificaciones, cámara, archivos, QR, identidad, ELIANA.

**Estado**: 🔴 No iniciado

## Fase 5: ZAFIRO AOSP

Arquitectura futura basada en Android Open Source Project: launcher personalizado, apps MSM preinstaladas, identidad ZAFIRO, tienda de aplicaciones autorizadas, configuración, seguridad, actualizaciones, perfiles de dispositivo.

**Estado**: 🔴 No iniciado (requiere build reproducible, imagen del sistema, dispositivo de prueba, boot verificado, OTA)

## Fase 6: ZAFIRO Living OS

Automatización contextual, ELIANA como orquestador, módulos adaptativos, dispositivos autorizados, telepresencia, gemelos digitales, sensores, ciudades y comunidades conectadas.
