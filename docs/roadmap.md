# Roadmap de producto

## Fase 0: Fundacion

Objetivo: dejar lista la base tecnica y de producto.

- Definir nombre final.
- Crear identidad visual inicial.
- Crear monorepo.
- Configurar Next.js, TypeScript, TailwindCSS y Supabase.
- Configurar CI/CD.
- Definir esquema inicial de base de datos.
- Activar autenticacion con Google y GitHub.
- Crear i18n espanol/ingles.

## Fase 1: MVP privado

Objetivo: validar si la experiencia pregunta + IA + comunidad funciona.

- Home con buscador central.
- Crear pregunta.
- Respuesta inicial de IA.
- Feed publico.
- Pagina de pregunta.
- Respuestas humanas.
- Comentarios.
- Votos.
- Perfil basico.
- Categorias.
- Busqueda basica.
- Guardar pregunta.
- Moderacion basica.
- Panel admin minimo.

## Fase 2: Beta publica

Objetivo: mejorar retencion, calidad y crecimiento organico.

- Traduccion automatica espanol/ingles.
- Busqueda semantica.
- Preguntas relacionadas.
- Resumen canonico.
- Seguimiento de preguntas, categorias y usuarios.
- Notificaciones in-app y push.
- Reputacion inicial.
- Insignias basicas.
- Reportes y cola de moderacion.
- SEO para preguntas publicas.

## Fase 3: Expertos

Objetivo: crear confianza y diferenciacion.

- Verificacion de especialidades.
- Validaciones de expertos.
- Perfiles profesionales.
- Ranking por especialidad.
- Respuestas destacadas.
- Correcciones verificadas.
- Fuentes y citas estructuradas.

## Fase 4: Plataforma

Objetivo: abrir ecosistema y monetizacion.

- Premium.
- Organizaciones verificadas.
- API publica.
- Marketplace de expertos.
- Marketplace de servicios.
- Cursos y certificaciones.
- Patrocinios por categoria.
- Analitica para organizaciones.

## Fase 5: Futuro

Objetivo: expandir el concepto hacia colaboracion avanzada.

- Agentes inteligentes.
- IA personalizada.
- Asistente por voz.
- Videollamadas.
- Comunidades.
- Universidades y empresas.
- Marketplace de inventos.
- Marketplace de proyectos e inversion.
- Realidad aumentada.
- Realidad virtual.
