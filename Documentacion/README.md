# Documentación del Imperio MSM

Este directorio centraliza la documentación estratégica, técnica y operativa del Imperio MSM.

## Estructura

- `Imperio_MSM_Plan_Maestro.pdf` — Plan Maestro del Imperio MSM
- `Imperio_MSM_Plan_Maestro.docx` — Versión editable del Plan Maestro
- `Arquitectura.pdf` — Diagramas de arquitectura del ecosistema
- `Roadmap.pdf` — Hoja de ruta y cronograma
- `Manual_Tecnico.pdf` — Documentación técnica para desarrolladores
- `Manual_OWNER.pdf` — Guía para el propietario del sistema
